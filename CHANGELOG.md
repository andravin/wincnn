# Changelog

## 2.0.0 — 2026-02-03

- Dropped Python 2 support; requires Python >= 3.8
- Requires SymPy >= 1.9
- Fixed SymPy deprecation warning for non-Expr objects in Matrix
- Added type hints and docstrings to public API
- Added `pyproject.toml` (PEP 621)
- Added pytest test suite
- Added GitHub Actions CI

## 1.0.0 — 2016-07-04

- Initial release
- Cook-Toom filter and convolution transform generation
- Symbolic verification of transforms
- Python 2.7 / SymPy 1.0 compatible
